# Renesas MCU Software Overall Offer

Welcome to the Renesas MCU software overview page.

This repository serves as an entry point to the public software resources, documentation, example projects, and support channels available for Renesas MCUs on GitHub and related Renesas platforms.

The goal of this page is to help developers quickly find the right starting point depending on the MCU family, software package, and development workflow they are using.

---

## What You Will Find Here

This page provides a structured overview of software resources for:

- **RA MCUs**
  - RA Flexible Software Package (FSP)
  - RA example projects
  - RA application projects
  - RA documentation and downloads

- **RX MCUs**
  - RX Driver Package
  - RX-related public software repositories
  - RX documentation and product resources

This page is intended as a navigation hub.  
It does **not** duplicate the source content of the linked repositories. Instead, it helps you locate the correct repository, download, or documentation page based on your development needs.

## RA MCU Software Offer

Renesas RA MCUs are supported by the **Flexible Software Package (FSP)**, which provides:

- Board support and peripheral drivers
- Middleware and RTOS integration
- Development support through e² studio and related tools
- Documentation and examples to accelerate evaluation and development

### Key RA resources

| Resource | Description | Link |

## Development Flow Guidance

### For RA MCU users
A typical RA development flow is:

1. Select the RA device or evaluation kit
2. Install/download the RA Flexible Software Package
3. Review the FSP documentation
4. Import or download example/application projects as needed
5. Build and evaluate in the supported tool environment

### For RX MCU users
A typical RX development flow is:

1. Select the RX device or evaluation kit
2. Review the RX product information
3. Obtain the required RX driver/software package
4. Use the appropriate development environment and package version for your project
